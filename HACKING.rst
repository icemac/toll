Run the tests
=============

Install tox_ as the test runner and the call it to run the tests::

    $ tox

To run the tests of `toll` using `toll` itself call::

    $ python3 -m venv .
    $ bin/pip install zc.buildout
    $ bin/buildout
    $ bin/toll

.. _tox : http://tox.readthedocs.io/en/latest/install.html
