Run the tests
=============

To run the tests install the test runner and call it::

    $ python3.5 bootstrap.py
    $ bin/buildout
    $ bin/py.test


Upload the docs
===============

Note to self how to upload the documentation::

    $ python3.5 setup.py upload_docs
